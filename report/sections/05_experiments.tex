\chapter{Experiments}

\section{Preliminary Experiment}

A first intermediate experiment was run to verify that the pipeline was correct and that the neural network was able to learn the pricing function.
The command used was:

\begin{lstlisting}[style=pythonstyle, caption={Command for a preliminary experiment.}]
python scripts/run_experiment.py \
  --n-samples 10000 \
  --max-epochs 40 \
  --batch-size 1024 \
  --mc-n-paths 5000 \
  --mc-evaluation-samples 128
\end{lstlisting}

This run does not necessarily represent the final experiment of the project; rather, it is an initial correctness check of the implementation.
Final experiments may use more observations, more training epochs, and a larger number of Monte Carlo paths.

\section{Planned Visualizations}

The main visualizations are:

\begin{itemize}
    \item training loss and validation loss;
    \item comparison between the Black-Scholes price and the neural network prediction;
    \item distribution of prediction errors;
    \item absolute error as a function of moneyness $S_0/K$;
    \item absolute error as a function of maturity $T$;
    \item absolute error as a function of volatility $\sigma$.
\end{itemize}
